Quickstart with Gemini CLI

Your AI-powered assistant, right in your computer's terminal.

First... What is a "CLI"?

Imagine you could talk to your computer by typing commands instead of clicking buttons. That's a Command Line Interface (CLI). It's a powerful and fast way for developers (and you!) to get things done. The Gemini CLI brings Google's powerful AI directly to that text-based interface.

Installation

Let's get Gemini CLI set up on your computer. It’s a simple process using npm, a tool that helps install software like this. First, you'll need to have Node.js and npm installed. You can get them from the official Node.js website.

For macOS

  1. Open "Terminal" (you can find it in Applications → Utilities).
  2. Copy the command below, paste it into your terminal, and press Enter. This will download and install the Gemini CLI package. 
    npm install -g @google/gemini-cli
  3. After installation, type gemini and press Enter. The CLI will guide you to log in with your Google account in your web browser. This is a one-time setup to connect the CLI to your account. 
    gemini

For Windows

  1. Open "PowerShell" (search for it in the Start Menu).
  2. Copy the command below, paste it into your terminal, and press Enter. This will download and install the Gemini CLI package. 
    npm install -g @google/gemini-cli
  3. After installation, type gemini and press Enter. The CLI will guide you to log in with your Google account in your web browser. This is a one-time setup to connect the CLI to your account. 
    gemini

That's it! You're ready to go. The next sections show you what you can do.

Awesome Examples

Here are a few ways you can start using Gemini CLI right away.

1. Chat with Gemini

This is the most basic use. Just ask it anything!

gemini "Explain what a black hole is in simple terms"

2. Understand Your Code

Have a file you don't understand? Ask Gemini to explain it. It can read files directly from your computer.

gemini "Explain what this code does and suggest improvements" -f "path/to/your/file.js"

3. Generate Code from a Prompt

Need a quick Python script or a React component? Describe what you want, and Gemini will build it for you.

gemini "Write a Python function that takes a list of numbers and returns only the even ones"
# Example Output: def get_even_numbers(numbers): """ Takes a list of numbers and returns a new list containing only the even ones. """ return [num for num in numbers if num % 2 == 0]

4. Organize Your Files

Let Gemini help you tidy up! For example, it can look at all the images in a folder and rename them based on what's in the picture.

gemini "For each image in this folder, rename it to describe the image content" -f "path/to/your/images/*"

Primero... ¿Qué es una "CLI"?

Imagina que puedes hablar con tu computadora escribiendo comandos en lugar de hacer clic en botones. Eso es una Interfaz de Línea de Comandos (CLI). Es una forma potente y rápida que usan los desarrolladores (¡y ahora tú!) para hacer tareas. La Gemini CLI trae la poderosa IA de Google directamente a esa interfaz de texto.

Instalación

Vamos a configurar Gemini CLI en tu computadora. Es un proceso simple usando npm, una herramienta que ayuda a instalar software como este. Primero, necesitarás tener Node.js y npm instalados. Puedes obtenerlos desde el sitio web oficial de Node.js.

Para macOS

  1. Abre "Terminal" (la encuentras en Aplicaciones → Utilidades).
  2. Copia el siguiente comando, pégalo en tu terminal y presiona Enter. Esto descargará e instalará el paquete de Gemini CLI. 
    npm install -g @google/gemini-cli
  3. Después de la instalación, escribe gemini y presiona Enter. La CLI te guiará para que inicies sesión con tu cuenta de Google en tu navegador web. Es una configuración única para conectar la CLI a tu cuenta. 
    gemini

Para Windows

  1. Abre "PowerShell" (búscalo en el Menú de Inicio).
  2. Copia el siguiente comando, pégalo en tu terminal y presiona Enter. Esto descargará e instalará el paquete de Gemini CLI. 
    npm install -g @google/gemini-cli
  3. Después de la instalación, escribe gemini y presiona Enter. La CLI te guiará para que inicies sesión con tu cuenta de Google en tu navegador web. Es una configuración única para conectar la CLI a tu cuenta. 
    gemini

Ejemplos Geniales

Aquí tienes algunas ideas para empezar a usar Gemini CLI ahora mismo.

1. Chatea con Gemini

Este es el uso más básico. ¡Simplemente pregúntale lo que sea!

gemini "Explica qué es un agujero negro en términos sencillos"

2. Entiende tu Código

¿Tienes un archivo de código que no entiendes? Pídele a Gemini que te lo explique. Puede leer archivos directamente de tu computadora.

gemini "Explica qué hace este código y sugiere mejoras" -f "ruta/a/tu/archivo.js"

3. Genera Código a partir de una Descripción

¿Necesitas un script rápido en Python o un componente de React? Describe lo que quieres y Gemini lo creará para ti.

gemini "Escribe una función en Python que reciba una lista de números y devuelva solo los pares"
# Ejemplo de Respuesta: def get_even_numbers(numbers): """ Recibe una lista de números y devuelve una nueva lista solo con los números pares. """ return [num for num in numbers if num % 2 == 0]

4. Organiza tus Archivos

¡Deja que Gemini te ayude a ordenar! Por ejemplo, puede ver todas las imágenes en una carpeta y renombrarlas según lo que haya en la foto.

gemini "Por cada imagen en esta carpeta, renuémbrala para describir el contenido de la imagen" -f "ruta/a/tus/imagenes/*"
