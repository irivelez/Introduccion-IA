Welcome to the Gemini CLI Guide!

This guide will walk you through installing and using the Gemini Command Line Interface (CLI). Think of the CLI as a way to chat with the Gemini AI directly from a text-based window on your computer. It's powerful, fast, and easier to set up than you think!

Part 1: Installation

Before we can use Gemini, we need to install it. This requires a program called Python. Let's go step-by-step.

Step 1: Get the Command Window (Terminal)

This is the special window where we'll type our commands.

  • Windows: Press the Windows key, type cmd or PowerShell, and press Enter.
  • macOS: Open Finder, go to Applications → Utilities, and double-click on Terminal.

Step 2: Check for Python

The Gemini CLI is a Python package, so you need Python installed. Let's see if you have it. Type this command in your terminal and press Enter:

python --version
Note: On some systems, especially macOS and Linux, you might need to use python3 instead of python. If the command above gives an error, try: python3 --version.

If you see a version number (e.g., "Python 3.9.1"), great! You can skip to Step 4. If not, don't worry—let's install it.

Step 3: (If Needed) Install Python

If you don't have Python, it's a quick install.

  1. Go to the official Python website: python.org/downloads.
  2. Click the big "Download Python" button. It will get the right version for your computer.
  3. Run the installer you downloaded.
  4. Important for Windows users: On the first screen of the installer, make sure to check the box that says "Add Python to PATH" before clicking "Install Now". This is very important!

Once installed, close and reopen your terminal window and try the command from Step 2 again to make sure it worked.

Step 4: Install Gemini CLI

This is the final installation step! With your terminal open, copy and paste the following command and press Enter. This tells Python's package manager (pip) to download and install Gemini.

pip install gemini-cli
Note: If you've been using python3, you may need to use pip3 here: pip3 install gemini-cli.

Step 5: Verify Your Installation

To make sure everything is working, ask Gemini for its version number:

gemini --version

If you see a version number, congratulations! You've successfully installed the Gemini CLI.

Part 2: Quickstart Examples

Now for the fun part. Using it is as simple as typing `gemini` followed by your question in quotes.

Example 1: Your First Conversation

Let's start with a simple question to make sure it's listening.

gemini "Hello! In one sentence, what are you?"
Example Output:

I am a large language model, trained by Google.

Example 2: Get Creative

Ask Gemini to write something for you. How about a haiku (a 3-line poem with a 5, 7, 5 syllable structure) about computers?

gemini "Write a haiku about a computer."
Example Output:

Screen glows in the dark,
Silent thoughts in circuits run,
A world at my keys.

Example 3: Learn Something New

Use it as a super-fast encyclopedia. Let's ask it to explain a technical topic in a simple way.

gemini "Explain what an API is like I'm five years old."
Example Output:

Imagine you're at a restaurant. You can't go into the kitchen to get your food, right? So you have a waiter (the API!). You tell the waiter what you want from the menu, the waiter goes to the kitchen, gets your food, and brings it back to you. You don't need to know how the kitchen works, just how to ask the waiter!

And that's it!

You're all set. You can now ask Gemini anything you want, right from your computer's terminal. Enjoy exploring!

¡Bienvenido a la Guía de Gemini CLI!

Esta guía te enseñará a instalar y usar la Interfaz de Línea de Comandos (CLI) de Gemini. Piensa en la CLI como una forma de chatear con la IA de Gemini directamente desde una ventana de texto en tu computadora. ¡Es potente, rápida y más fácil de configurar de lo que crees!

Parte 1: Instalación

Antes de que podamos usar Gemini, necesitamos instalarlo. Esto requiere un programa llamado Python. Vayamos paso a paso.

Paso 1: Abrir la Ventana de Comandos (Terminal)

Esta es la ventana especial donde escribiremos nuestros comandos.

  • Windows: Presiona la tecla de Windows, escribe cmd o PowerShell, y presiona Enter.
  • macOS: Abre Finder, ve a Aplicaciones → Utilidades, y haz doble clic en Terminal.

Paso 2: Verificar si tienes Python

Gemini CLI es un paquete de Python, por lo que necesitas tener Python instalado. Veamos si lo tienes. Escribe este comando en tu terminal y presiona Enter:

python --version
Nota: En algunos sistemas, especialmente en macOS y Linux, podrías necesitar usar python3 en lugar de python. Si el comando anterior da un error, intenta: python3 --version.

Si ves un número de versión (ej. "Python 3.9.1"), ¡genial! Puedes saltar al Paso 4. Si no, no te preocupes, vamos a instalarlo.

Paso 3: (Si es necesario) Instalar Python

Si no tienes Python, la instalación es rápida.

  1. Ve al sitio web oficial de Python: python.org/downloads.
  2. Haz clic en el botón grande que dice "Download Python". Descargará la versión correcta para tu computadora.
  3. Ejecuta el instalador que descargaste.
  4. Importante para usuarios de Windows: En la primera pantalla del instalador, asegúrate de marcar la casilla que dice "Add Python to PATH" antes de hacer clic en "Install Now". ¡Esto es muy importante!

Una vez instalado, cierra y vuelve a abrir tu ventana de terminal e intenta el comando del Paso 2 de nuevo para asegurarte de que funcionó.

Paso 4: Instalar Gemini CLI

¡Este es el último paso de la instalación! Con tu terminal abierta, copia y pega el siguiente comando y presiona Enter. Esto le dice al gestor de paquetes de Python (pip) que descargue e instale Gemini.

pip install gemini-cli
Nota: Si has estado usando python3, puede que necesites usar pip3 aquí: pip3 install gemini-cli.

Paso 5: Verificar tu Instalación

Para asegurarte de que todo funciona, pregúntale a Gemini su número de versión:

gemini --version

Si ves un número de versión, ¡felicidades! Has instalado correctamente Gemini CLI.

Parte 2: Ejemplos de Inicio Rápido

Ahora la parte divertida. Usarlo es tan simple como escribir `gemini` seguido de tu pregunta entre comillas.

Ejemplo 1: Tu Primera Conversación

Comencemos con una pregunta simple para asegurarnos de que está escuchando.

gemini "¡Hola! En una oración, ¿qué eres?"
Ejemplo de Respuesta:

Soy un modelo de lenguaje grande, entrenado por Google.

Ejemplo 2: Ponte Creativo

Pídele a Gemini que escriba algo para ti. ¿Qué tal un haiku (un poema de 3 versos con una estructura de 5, 7, 5 sílabas) sobre computadoras?

gemini "Escribe un haiku sobre una computadora."
Ejemplo de Respuesta:

Brilla la pantalla,
Ideas en circuitos fluyen,
Un mundo a mis pies.

Ejemplo 3: Aprende Algo Nuevo

Úsalo como una enciclopedia súper rápida. Pidámosle que explique un tema técnico de una manera sencilla.

gemini "Explica qué es una API como si yo tuviera cinco años."
Ejemplo de Respuesta:

Imagina que estás en un restaurante. No puedes entrar en la cocina a buscar tu comida, ¿verdad? Para eso tienes un mesero (¡la API!). Le dices al mesero lo que quieres del menú, el mesero va a la cocina, busca tu comida y te la trae. ¡No necesitas saber cómo funciona la cocina, solo cómo pedirle al mesero!

¡Y eso es todo!

Ya estás listo. Ahora puedes preguntarle a Gemini lo que quieras, directamente desde la terminal de tu computadora. ¡Disfruta explorando!
