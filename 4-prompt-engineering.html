✨ "Good morning, how are you?"

Básico
📚 Few-Shot Prompting
Aprendizaje por ejemplos
Incluye 2-5 ejemplos para mostrar el patrón deseado. Mejora dramáticamente la precisión del modelo.
❌ Sin Ejemplos
"Clasifica el sentimiento de: 'El producto llegó roto pero el servicio al cliente fue excelente'"

Resultado:

😕 "Positivo" o "Negativo" - Respuesta confusa para sentimientos mixtos

Impreciso Binario
✅ Con Ejemplos
Clasifica el sentimiento: Ejemplo 1: "Excelente producto" → Positivo Ejemplo 2: "Terrible experiencia" → Negativo Ejemplo 3: "Buena calidad pero caro" → Mixto Clasifica: "El producto llegó roto pero el servicio al cliente fue excelente"

Resultado:

✨ "Mixto - El producto tiene problemas (roto) pero el servicio fue positivo (excelente). Aspectos negativos: calidad del producto. Aspectos positivos: atención al cliente."

Preciso Matizado Aprende patrón
Intermedio
🧠 Chain-of-Thought
Razonamiento paso a paso
Guía al modelo para que muestre su razonamiento. Esencial para problemas complejos y matemáticos.
❌ Directo
"Si una tienda tiene 40% de descuento en un producto de $150 y luego 20% adicional, ¿cuál es el precio final?"

Resultado:

😕 "$72" - Respuesta sin mostrar cálculos, posible error

Sin razonamiento Posible error
✅ Chain-of-Thought
Resuelve paso a paso: Si una tienda tiene 40% de descuento en un producto de $150 y luego 20% adicional, ¿cuál es el precio final? Piensa paso a paso: 1. Identifica el precio original 2. Calcula el primer descuento 3. Calcula el segundo descuento sobre el precio ya rebajado 4. Determina el precio final

Resultado:

✨ 1. Precio original: $150
2. Primer descuento (40%): $150 × 0.40 = $60
   Precio después del primer descuento: $150 - $60 = $90
3. Segundo descuento (20% sobre $90): $90 × 0.20 = $18
   Precio después del segundo descuento: $90 - $18 = $72
4. Precio final: $72

Específico Experto Actionable
Avanzado
🔄 Self-Consistency
Múltiples caminos de razonamiento
Genera múltiples soluciones y elige la más consistente. Ideal para problemas donde la precisión es crítica.
❌ Una Sola Respuesta
"Una empresa reduce su personal en 25%. Si tenía 120 empleados, ¿cuántos tiene ahora?"

Resultado:

😕 "95 empleados" - Error potencial sin verificación

Sin verificación Propenso a errores
✅ Self-Consistency
Una empresa reduce su personal en 25%. Si tenía 120 empleados, ¿cuántos tiene ahora? Resuelve esto de 3 maneras diferentes: 1. Calculando la reducción directamente 2. Calculando el porcentaje que permanece 3. Usando proporciones Luego compara los resultados para verificar la respuesta correcta.

Resultado:

✨ Método 1 - Reducción directa:
25% de 120 empleados = 0.25 × 120 = 30 empleados despedidos
Empleados restantes: 120 - 30 = 90 empleados

Método 2 - Porcentaje que permanece:
Si despiden 25%, permanece 75%
75% de 120 = 0.75 × 120 = 90 empleados

Método 3 - Proporciones:
120 empleados = 100%
X empleados = 75% (100% - 25%)
X = (120 × 75) ÷ 100 = 90 empleados

✅ Los 3 métodos confirman: 90 empleados restantes

Verificado Múltiples métodos Confiable
Avanzado
🌳 Tree-of-Thoughts
Exploración de múltiples ramas
Explora sistemáticamente diferentes enfoques como un árbol de decisiones. Para problemas complejos con múltiples soluciones.
❌ Enfoque Linear
"¿Cómo puedo aumentar las ventas de mi restaurante?"

Resultado:

😕 Lista simple de sugerencias generales sin análisis profundo de opciones

Superficial Sin análisis
✅ Tree-of-Thoughts
Ayúdame a aumentar las ventas de mi restaurante. Explora estas 3 ramas principales: RAMA 1: Mejorar experiencia del cliente - Evalúa: servicio, ambiente, calidad - Pros/contras de cada mejora - Costo vs impacto RAMA 2: Estrategias de marketing - Evalúa: digital, tradicional, boca a boca - Presupuesto necesario para cada - ROI esperado RAMA 3: Optimización operativa - Evalúa: menú, precios, horarios - Análisis de competencia - Implementación gradual Para cada rama, dame tu recomendación final.

Resultado:

✨ ANÁLISIS TREE-OF-THOUGHTS:

RAMA 1: Experiencia del Cliente
Evaluación:
• Servicio: Capacitar personal (+), costo moderado (-)
• Ambiente: Renovar decoración (+), inversión alta (-)
• Calidad: Mejorar ingredientes (+), aumenta costos (-)
Costo vs Impacto: Medio/Alto
ROI esperado: 15-25% incremento en 6 meses

RAMA 2: Marketing
Digital: Instagram/TikTok (+), requiere constancia (-)
Tradicional: Radio local (+), difícil medir ROI (-)
Boca a boca: Programa referidos (+), crecimiento lento (-)
Presupuesto: $500-2000/mes según canal
ROI esperado: 20-40% nuevos clientes

RAMA 3: Optimización Operativa
Menú: Platos populares (+), eliminar poco vendidos (+)
Precios: Análisis competencia, ajuste estratégico
Horarios: Extender fines de semana (+), costos laborales (-)
Implementación: Inmediata, bajo costo

🎯 RECOMENDACIÓN FINAL:
1. Prioridad ALTA: Rama 3 (optimización) - implementar inmediatamente
2. Prioridad MEDIA: Rama 2 (marketing digital) - iniciar en paralelo
3. Prioridad BAJA: Rama 1 (renovaciones) - planificar para siguiente trimestre

Justificación: Optimización operativa ofrece ROI inmediato con menor inversión, mientras marketing digital construye base de clientes futura.

💡 Principios Fundamentales

🎯 Especificidad es Clave
Cuanto más específico seas, mejores resultados obtienes. "Traduce esto" vs "Traduce del español al inglés formal"
📋 Define el Formato
Especifica cómo quieres la respuesta: lista numerada, párrafos, tabla, JSON, etc. El formato importa tanto como el contenido.
🎭 Usa Roles Específicos
"Actúa como un [experto específico]" produce respuestas más relevantes que preguntas genéricas.
📚 Ejemplos son Poderosos
2-3 ejemplos bien elegidos pueden ser más efectivos que páginas de instrucciones detalladas.
🔄 Itera y Refina
El primer prompt rara vez es perfecto. Experimenta, ajusta y mejora basándote en los resultados obtenidos.
🧠 Guía el Razonamiento
Para problemas complejos, pide al modelo que "piense paso a paso" o muestre su razonamiento antes de la respuesta final.

⚡ Tips Rápidos para Mejores Prompts

🎯 Contexto Primero Proporciona toda la información relevante antes de hacer la pregunta
📏 Controla la Longitud Especifica si quieres respuestas cortas, detalladas o de longitud específica
🚫 Evita Negaciones Di lo que QUIERES, no lo que NO quieres. Es más claro y efectivo
🔢 Usa Delimitadores Separa secciones con """, ---, ### para mayor claridad y estructura
📊 Pide Verificación Para información importante, pide al modelo que cite fuentes o muestre confianza
⏰ Considera el Tiempo Menciona si necesitas información actual o específica de cierta época
